    보던거 이어봄. 아파치 설치하고 아이피주소/index.html 쳐서 확인한다.
    IP 주소 = Internet Protocol Address
    웹브라우저가 설치되어 있는 컴퓨터를 가리키는 특수한 주소, 약속된 주소
    윈도우 + R 에다 cmd를 켜면 창이 뜸
    여기에 ping naver.com 치면 숫자가 떠욤 (하고 ctrl+c하면 꺼짐)
    125.209.222.141 은 네이버의 컴퓨터를 가리키는 IP 주소
    사람은 이름처럼 도메인을 쓰더라도 기계끼리는 전화번호와 같은 IP주소로 통신한다

    htdoc 에 있는 파일들을 삭제하고 내가 만든 것들을 붙여넣어서 리로드하면 바뀌지?
    내가 만든 웹페이지에 서버가 생긴거야
    http = hypertext transfer Protocol
    = 웹페이지 전송 약속/규약
    = 웹페이지를 웹서버와 브라우저가 주고받기 위한 약속
    우리처럼 한 대의 컴퓨터 안에 서버 브라우저 index.html파일이 다 있는 경우는 상관없지만
    서로 다른 컴퓨터 사이에 웹페이지를 주고받을 떄는 http를 사용해야 한다.
    우리가 웹서버의 IP주소를 얻어냈다면 http://아이피주소/index.html 을 입력하면 들어갈 수 있는데
    시작표시줄 아이콘에 무선이나 유선 오른쪽클릭해서 자세히로 들어가면 IPv4 주소라고 뜬다 - 이것이 현재 내가 쓰고 있는 컴퓨터의 IP주소를 보여주는 것
    스마트폰과 컴퓨터가 같은 네트워크에 접속해 있어야 함
